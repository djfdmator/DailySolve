# DailySolve 

### 풀이 프리뷰

1. 피라미드의 맨 위에서 부터 아래로 내려오며 계산한다.
2. 계산한 결과 값들을 따로 저장해둔다.
3. 저장해둔 결과값들과 기존 피라미드에 있던 값들을 이용해서 다음 행의 값들을 계산한다.
