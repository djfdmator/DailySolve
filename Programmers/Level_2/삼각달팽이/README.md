# DailySolve 

### 풀이 프리뷰

1. 달팽이를 그리는 순서에 맞게 배열을 순회할 수 있도록 한다.
2. 달팽이가 들어갈 수 있는 크기의 배열을 준비한다.
3. 1번으로 배열을 순회하며 값을 넣는다.
4. 값을 넣고 다음 순회할 곳이 배열의 크기를 안넘고 값이 아직 안들어간 곳이면 인덱스를 넘기고 값을 증가시킨다.
5. 다음 순회할 곳이 배열의 크기를 넘거나 이미 값이 들어가 있다면 순회 방향을 다음으로 바꿔준다.
6. 3~5를 넣는 값이 넣어야할 최댓값이랑 같아 질때까지 반복한다.
